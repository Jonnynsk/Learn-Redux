# Learn-Redux

## Redux

**Redux** - библиотека для JavaScript с открытым исходным кодом, предназначенная для управления состоянием приложения. Чаще всего используется в связке с React для разработки клиентской части. В основе лежит однонаправленный поток данных. Это означает, что любое действие в приложении обрабатывается по единому сценарию — в одну сторону. Синхронный по-умолчанию.
